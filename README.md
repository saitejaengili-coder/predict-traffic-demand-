# Traffic Demand Prediction using LightGBM & XGBoost

## Project Overview
This project predicts future traffic demand using machine learning. The model is trained on traffic, weather, and time-based features and deployed using Flask.

## Features
- Data Preprocessing
- Feature Engineering
- LightGBM
- XGBoost
- Random Forest
- Flask Deployment
- REST API

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- XGBoost
- Flask

## Dataset Features
- Hour
- Day
- Temperature
- Humidity
- Rainfall
- Speed Limit
- Traffic Density

## Run

Install dependencies

```bash
pip install -r requirements.txt
```

Train Model

```bash
python train_model.py
```

Run Flask

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
